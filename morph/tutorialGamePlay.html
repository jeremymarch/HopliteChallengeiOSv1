Game Play


  • Change the given form to the form indicated; you will have 30 seconds.
  • If the indicated form has multiple forms, give them all. Enter multiple forms by pressing the MF key.
    • The first press will add 50% more time to the timer to account for the extra difficulty.
    • Pressing the MF button again will type a comma to separate the forms.
  • Forms are described by their morphology, e.g. the deponent ἐβουλήθην which is active in meaning, but passive in morphology, is described as passive.
  • Correct forms earn points and a new change will be asked using the entered form as the starting point. Three forms correct from a single verb will earn bonus points and play resumes with a new verb.
  • Incorrect forms will cost a life; the correct form will be displayed at the below. Press Continue and play will resume with a new verb.
  • You start the game with three "lives," indicated in the top right corner by the symbols. The game continues until you lose all of your lives.
  • Before the start of each game you can select which Hansen & Quinn units you would like to include in your game. From the Main Menu go to Settings to turn on or off units. In Play mode the game will include any verb from the highest selected unit and under, so selecting unit 5 will also include units 1-4. In practice mode the app will include verbs only from the units selected, so you can concentrate on verbs from just one unit for extra practice.
  • When only units 1 and/or 2 are selected, only one change will be made to each verb form. When a higher unit is selected, two changes will be made for each new form.
  • For each correctly conjugated form you will receive points equal to the highest selected unit, e.g. if the highest selected unit is 19, you will receive 19 points per correct answer. By answering three changes to one verb correctly you will receive a bonus equal to the higest unit.
  • The game does not currently ask for any periphrastic forms, these will be added in a later version.
  • The game does not currently ask for the base form of verbs which require a prefix such as -ἔβην.
