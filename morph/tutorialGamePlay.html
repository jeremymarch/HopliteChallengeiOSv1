How to Play


Game Play
  • Change the given form to the form indicated; you will have 30 seconds.
  • Correct forms earn points and a new change will be asked using the entered form as the starting point. After four correct forms from the same verb, a bonus is awarded and play resumes from a new verb.
  • Incorrect forms will cost a life; the correct form will be displayed below. Press Continue to resume play with a new verb.
  • You start the game with three lives, indicated in the top right corner by this symbol . The game continues until you lose all of your lives.
  • Forms are described by their morphology. For example, the deponent ἐβουλήθην which is active in meaning, but passive in morphology, is described as passive.
H&Q Units
  • Before the start of each game you can select which Hansen & Quinn units you would like to include in your game. From the Main Menu go to Settings to turn on or off units.
    • In Play mode the game will include any verb from the highest selected unit and under, so selecting unit 5 will also include units 1-4.
    • In practice mode the app will only include verbs from the units selected, so you can concentrate on verbs from just one unit for extra practice.
  • When only units 1 and/or 2 are selected, only one change will be made to each verb form. When a higher unit is selected, two changes will be made for each new form.
  • The game does not currently ask for periphrastic forms.
  • The game does not currently ask for the base form of verbs which require a prefix such as -ἔβην.
Multiple Forms
  • If the indicated form has multiple forms, give them all. Enter multiple forms by pressing the MF key.
    • The first press will add 50% more time to the timer to account for the extra difficulty.
    • Pressing the MF button again will type a comma to separate the forms.
Scoring
  • For each correctly conjugated form you will receive points equal to the highest selected unit. For example, if the highest selected unit is 19, you will receive 19 points per correct answer. After entering four correct forms from the same verb, you will receive bonus points equal to the highest selected unit.
