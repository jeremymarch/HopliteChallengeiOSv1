Using the Keyboard


  • Accents and other diacritics should be typed after their vowel. If a vowel has more than one diacritic they can be entered in any order. Pressing the same diacritic key again toggles off the diacritic; pressing a mutually exclusive diacritic will replace the original diacritic with the second, so pressing the circumflex for a vowel which already has an acute replaces the acute accent with a circumflex. Don't forget those macrons!
  • Press the MF button if there are multiple forms. The first press gives you 50% more time, subsequent presses add commas to separate the forms. If there is only one form, though, pressing MF will immediately result the attempt being marked incorrect, so use it with caution!
  • For a nu-movable, press the nu key followed by the () key.
  • Final sigma is in the bottom row beneath kappa.
  • You can move the cursor back in the word to add an accent or make a correction by holding your finger over the word for a couple of seconds. A magnifying lense is displayed to help you place the cursor.
